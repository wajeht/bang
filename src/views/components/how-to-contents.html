Setting Up Your Browser

Before you can use Bang commands from your browser's search bar, add Bang as a custom search engine:

  1. In your browser settings, add a new search engine:

    • Name: Bang (or whatever you prefer)
    • Shortcut: bd (or any keyword you prefer)
    • URL: http://bang.jaw.dev/?q=%s

  2. Now you can use Bang directly from your browser's address bar!


Browser-Specific Setup Instructions

  • Chrome: Settings → Search Engines → Manage Search Engines → Add
  • Firefox: Bookmarks → Manage Bookmarks → Right-click → New Bookmark → Add %s in URL
  • Edge: Settings → Privacy, search, and services → Address bar and search → Manage search engines → Add

Default Commands

  • @a, @action, @actions - Quick access to your actions
  • @n, @note, @notes - Quick access to your notes
  • @bm, @bookmark, @bookmarks - Quick access to your bookmarks
  • @api - Go to Swagger API docs
  • @b, @bangs - Go to home page
  • @s, @settings - Access settings
  • @data - Access data
  • @admin - Access admin

Special Commands

  • @bm [search term] - Search your bookmarks

    @a [search term] - Search your actions

    @n [search term] - Search your notes

    !bm [url] or !bm [title] [url] - Add a bookmark

    • [title] is optional - if not provided, we'll automatically fetch the page title
    • Example: !bm this is my bookmark title https://example.com

  • !add [trigger] [url] - Create a custom bang

    • Example: !add jaw https://bang.jaw.dev

  • !note [title] | [content] - Create a note

    • [title] is optional - if not provided, we'll use "Untitled"
    • Example with title: !note Important | Remember to check this website later
    • Example without title: !note This is a note without a title
